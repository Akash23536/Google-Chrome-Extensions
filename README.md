# Google-Chrome-Extensions
## 1) Puppet-Jokes English ->
Simple chrome extension the show English jokes whenever user click on it or reload the page 
and a refresh button is also added that fetch new jokes when user click on the button 
1. icanhazdadjoke API is used for fetching the random English jokes
## Published on Chrome Web Store 
link - https://chromewebstore.google.com/detail/puppet-jokes-english/ogoldblhjbamhclicnolkhaadajedlgp


## 1) Ha Ha Jokes  ->
Simple chrome extension the show English & Hindi jokes whenever user click on it or reload the page 
and English & Hindi buttons is added that display the jokes as per the selected language
1. icanhazdadjoke API is used for fetching the random English joke
2. hindi-jokes-api.onrender.com/ API is used for fetching the random Hindi jokes
## Published on Chrome Web Store 
link - https://chrome.google.com/webstore/detail/ha-ha-jokes/lekkmhcmkidhhohgeimcjkpmhbfjhkid
